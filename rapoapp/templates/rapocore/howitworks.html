How it works
This is how it works....
  1. Member A decides to let a book go from his/her shelf
  2. A releases the book on our website
  3. Member C requests for the book on our website
  4. Member B requests for the book too
  5. A gets C's address though our facebook group and sends the book physically to C
  6. A updates the physical act of sending on our website
  7. C receives the book physically
  8. C updates the receipt of the book on our website
  9. C reads the book. There is no time limit for this activity
  10. Once C is done with reading, C does a pass-on on our website to show that the book is available
  11. In this case, since B has already requested for the book, C sends the book to B by doing steps from 5
  12. The journey of the book continues ...
